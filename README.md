# 🛡 CyberShield - Cybersecurity Awareness Platform

## 📌 Overview

CyberShield is a full-stack cybersecurity awareness platform designed to educate users about cybersecurity threats and best practices.

## 🚀 Features

* User Authentication (Login/Register)
* Cybersecurity Learning Modules
* Interactive Security Quiz
* Password Strength Checker
* Phishing Email Detector
* Phishing Simulator
* SOC Log Analyzer
* Security Score Calculator
* User Dashboard
* PDF Certificate Generator

## 🛠 Tech Stack

### Frontend

* React.js
* React Router DOM
* CSS3

### Backend

* Node.js
* Express.js

### Libraries

* jsPDF
* Chart.js
* React ChartJS
